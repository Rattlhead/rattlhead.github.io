Вывод данных
----
Для вывода данных используется команда print(""). Любой текст предназначенный для "Человека" пишется в " или ', программа его не обрабатывает и выводит в том же виде.

<pre><code data-language="python">
#пример использования Print()
print("Это моя первая программа")
</code></pre>

<pre><code data-language="python">
#Вывод
>>Это моя первая программа
</code></pre>

Арифметические действия (операторы)
----
Но также в ней можно написать сразу код, который выполнится и мы увидим итог его выполнения. Для примера возьмем арифметические действия классические + - * /, а также рассмотрим несколько необычных ** // %

<pre><code data-language="python">
#Сложение
print(2 + 2) #4
#Вычитание
print(6 - 2) #4
#Умножение
print(4 * 4) #16
#Деление
print(9 / 3) #3.0
print(10 / 3)#3.33

#почему ответ получился 3.0, а не просто 3 мы разберем чуть позже.
</code></pre>

Теперь разберем с вами более сложные операторы.

<pre><code data-language="python">
#Возведение в степень
print(6 ** 2)  # Возводим число 6 в степень 2. Результат - 36

#Остаток от деления
print(10 % 3) # Ответ 1. Ближайшее число к 10, которое делится на 3 без остатка, это 9. Поэтому остаток от деления равен 10 - 9 = 1

#Целочисленное деление
print(7 / 2)  # 3.5
print(7 // 2) # Ответ 3, оператор // отбрасывает дробную часть
#почему ответ получился 3.0, а не просто 3 мы разберем чуть позже.

</code></pre>

Ввод данных
----
Ввод данных достаточно обширное понятие, данные можно получить от сканера, камеры, клавиатура, файла, мышки или страницы web-сайта. В нашем случае мы будет разбирать ввод данных в программу с помощью клавиатуры.

Для ввода используется команда input(), она ставит программу на паузу и ждет пока пользователь введет данные, после чего программа продолжает свое выполнение. Сама по себе функция не даст вам никакого результата, но вы и сами можете это проверить. Напишите программу с одной командой input() и посмотрите результат ее выполнения.

<pre><code data-language="python">
#Пример работы
input()
</code></pre>

Input позволяет принять данные пользователя, но на этом ее возможности заканчиваются. Для хранения, обработки и вывода этих данных используются переменные о которых мы сейчас и поговорим.

Переменные
----
Переменная - это листок с определенным именем, на который вы можете записать любую информацию, затем в любой момент эту информацию прочитать, изменить, удалить.
Переменные жизненно необходимо любой программе, без них вы не сможете ни записать настройки, ни имя пользователя. И в целом без них команда input() не имеет смысла. Давайте же разберемся как они работают подробнее на примере простой программы.

<pre><code data-language="python">
# Узнать у пользователя его имя и поприветствовать его
print ("Как тебя зову?")	# Выводим текст
name = input()						# Ждем ввод текста от пользователя
print ("Привет, " + name)	# Выводим текст с переменной, в которой храним имя
</code></pre>

А теперь давайте познакомимся с дополнительными возможностями команд input() и  print(). Для удобства мы можем не использовать лишнюю команду print, выведя тот же самый текст в input. А также команду print напишем более корректно и удобно.

<pre><code data-language="python">
# Узнать у пользователя его имя и поприветствовать его
name = input("Как тебя зову? ")							# Выводим текст и ждем ввод текста от пользователя
print (f"Привет, {name}, рад тебя видеть")	# Выводим текст с переменной, в которой храним имя
#Команда print(f"Текст {переменна}") позволяет удобно и читабильно, написать строку любого размера и сложности.
</code></pre>

Типы переменных
----
Теперь когда мы разобрались зачем нужны переменные, давай посмотрим какие они бывают и в чем их особенность.
В Python есть 4 примитивных типа данных:

<pre><code data-language="python">
# int (целые числа)
age = 18

# float (дробные числа)
fraction = 2.5

# str (строки)
fruit = 'apple'

# bool (правда или ложь)
isReady = True # всего два значения: True и False

</code></pre>

Правила написания переменных, как можно и как нельзя делать. В питоне принят стандарт PEP8.
Питон не требует указания типа переменой, он сам определяет в зависимости от введенных в нее данных.

* имя переменной может состоять только из букв, цифр и знака подчёркивания;
* имя не может начинаться с цифры;
* имя не может содержать специальных символов @, $, %.

Для название переменных принято использовать один из трех методов
* kebab-case — составные части переменной разделяются дефисом (my-super-var)
* snake_case — для разделения используется подчеркивание (my_super_var)
* CamelCase — каждое слово в переменной пишется с заглавной буквы (MySuperVar)

<pre><code data-language="python">
# Правильно
age = 19
year = '2000' # некорректно записывать число в " , но программа работать будет
name = 'Ivan'
my_name = "Maxim"
MyName = 'Vity'
my-name = "Alex"
isActive = True

# name и Name - это разные переменны
name = 'Petr'
Name = 'Ivan'
# name и Name - это разные переменны

# Неправильно
12name = 'Ivan'
!age = 12
город = "Moscow"

</code></pre>
